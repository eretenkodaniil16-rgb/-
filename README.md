# Сайт открытых учебных материалов

Статический сайт для GitHub Pages.

## Структура

```text
index.html
assets/
  styles.css
  script.js
documents/
  embryology_methodichka.pdf
.nojekyll
```

## Как обновлять

- Тексты и разметка: `index.html`
- Дизайн: `assets/styles.css`
- Логика сайта: `assets/script.js`
- PDF и методички: `documents/`

## Публикация через GitHub Pages

В настройках репозитория:

- Source: Deploy from a branch
- Branch: main
- Folder: / root

## Важно

HTML-теги не переводить. Они должны оставаться на английском: `<head>`, `<body>`, `<section>`, `<script>`, `<link>`.
